# Article Management System

A simple web application built to manage articles, featuring a metrics dashboard and full CRUD functionality using interactive modals.

### Technologies Used:
* **Framework:** Laravel 11
* **Frontend:** Bootstrap 5 & Blade Templates
* **Alerts & Modals:** SweetAlert2

---

## Setup Instructions

Follow these steps to run the project locally on your machine:

Quick Setup Instructions:
1. Run composer install and npm install from the project root
2. Configure local .env
3. Run php artisan migrate:fresh --seed to automatically build tables and populate them
4. Compile assets and start the local environment via npm run serve and php artisan serve
